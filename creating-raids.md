# Creating Raids

This guide explains how to create a raid on BandAid.

## What is a Raid?

A raid is a fundraising campaign for a Community Takeover (CTO). Creators set a target amount and deadline, and the community can donate to help reach the goal.

## Creating a Raid

{% stepper %}
{% step %}
### Connect Your Wallet

Click "Connect Wallet" and connect your Solana wallet.
{% endstep %}

{% step %}
### Fill Out the Raid Form

| Field                | Description             | Requirements                   |
| -------------------- | ----------------------- | ------------------------------ |
| **Contract Address** | The token's CA          | Required, valid Solana address |
| **Ticker**           | Token symbol            | Required, max 10 characters    |
| **Title**            | Raid title              | Required, max 50 characters    |
| **Description**      | Raid details            | Optional, max 250 characters   |
| **Raid Goal**        | What you're raising for | Optional, max 100 characters   |
| **Twitter Link**     | Social proof            | Optional, max 200 characters   |
| **Target Amount**    | SOL needed              | 0.1 - 1000 SOL                 |
| **Deadline**         | When raid ends          | 1 hour to 30 days from now     |
{% endstep %}

## Tips for Success

* Set realistic targets - Start small to build trust
* Provide social proof - Link your Twitter/community
* Be transparent - Clearly explain how funds will be used
* Engage community - Share progress and updates
